What is JACKY?

JACKY is a database containing information on 535 human kinases, 243,542 phosphosites, 11,477 human kinase-phosphosite relationships, 1,062 phosphosite-disease relationships, 1,465 human kinase inhibitors, and 60,057 inhibitor-kinase relationships. Kinases are enzymes that catalyze the transfer of phosphate groups from ATP to specific substrates/phosphosites located on the same kinase or a different molecule (generally a protein), in a process called phosphorylation. Since kinases are key regulators of most cell signalling networks, kinase mutations often result in diseases such as cancer, making them a target for therapy through the use of kinase inhibitors. JACKY was built by retrieving data from the following websites:

Kinases: UniProtKB , PDB

Phosphosites: Phospho.ELM, PhosphoSitePlus , (please note that PhosphoSitePlus data is not for commercial use, as per the terms and conditions on their website; navigate to Downloads > Datasets from PSP) ,

Kinase Inhibitors: BindingDB, chEMBL UniChem

Using JACKY

Search a kinase, phosphosite, or kinase inhibitor

Select whether you want to search a kinase, a phosphosite or a kinase inhibitor on the dropdown select menu on the top right corner of this page. Kinases can be searched by their protein name (e.g. KAPCA_HUMAN) or their gene name (e.g. PRKACA). If you select phosphosite, another dropdown select menu appears, giving you the option to search a phosphosite by the protein they're in, (one of) the kinase(s) by which they are phosphorylated, or their neighbouring sequence, as well as the option to access the phosphosite genome browser.

Phosphosite Genome Browser (PGB)

The PGB gives you the option to see which phosphosites have been mapped to a specific chromosomal location. You can select a whole chromosome or a specific g-band. You can access it by selecting 'Phosphosite' on the select dropdown menu on the top right corner of this page, and then 'Genome Location' on the select dropdown menu that will appear next to the first one.

Phosphoproteomics Data Analysis

Submit a tsv file (other extensions won't work) containing phosphoproteomics data by clicking on the 'Choose File' button of this page.

The file should follow the following format. The correct ordering of the columns is essential for the analysis to work, though the column headers do not matter. For one inhibitor, follow the format below, noting that the last two columns are optional. For multiple inhibitors, columns 1-2 must only be present once, columns 6-7 are optional, and columns 3-5 (or 3-7) should be repeated once for each inhibitor.

Each substrate in the file is matched to the kinase that phosphorylates it by searching JACKY, and the relative activity score (RAS) of each kinase is calculated. This information is returned as four tables and two figures, all of which can be downloaded. The first table will appear on the website, and shows the RAS of the 10 most enhanced and 10 most inhibited kinases, as well as the log fold change, p-value and number of substrates:

The full version, listing all kinases, can be downloaded. The kinases' RAS are also shown on a bar plot.

The second table shows each substrate on the file submitted and the kinase that phosphorylates them, as well as other information on each phosphosite, For some substrates, a kinase match may not be found on JACKY. The third table has the same columns as the second one, but contains only the substrates for which a kinase match was not found, while the fourth table contains only the substrates for which a kinase match was found.

A volcano plot shows the substrates plotted according to their log fold change (x axis) and their negative log10p (y axis).
