What is JACKY?

JACKY is a database containing information on 535 human kinases, 243,542 phosphosites, 11,477 human kinase-phosphosite relationships, 1,062 phosphosite-disease relationships, 30,016 human kinase inhibitors, and 58,978 inhibitor-kinase relationships. Kinases are enzymes that catalyze the transfer of phosphate groups from ATP to specific substrates/phosphosites located on the same kinase or a different molecule (generally a protein), in a process called phosphorylation. Since kinases are key regulators of most cell signalling networks, kinase mutations often result in diseases such as cancer, making them a target for therapy through the use of kinase inhibitors. JACKY was built by retrieving data from the following websites:

Kinases: UniProtKB , PDB

Phosphosites: PhosphoSitePlus please note that phosphosite.org data is not for commercial use , Phospho.ELM

Kinase Inhibitors: KIDFamMap

Using JACKY

Search a kinase, phosphosite, or kinase inhibitor

Select whether you want to search a kinase, a phosphosite or a kinase inhibitor on the dropdown select menu on the top right corner of this page. Kinases can be searched by their protein name (e.g. KAPCA_HUMAN) or their gene name (e.g. PRKACA). If you select phosphosite, another dropdown select menu appears, giving you the option to search a phosphosite by the protein they're in, (one of) the kinase(s) by which they are phosphorylated, or their neighbouring sequence, as well as the option to access the phosphosite genome browser.

Phosphosite Genome Browser (PGB)

The PGB gives you the option to see which phosphosites have been mapped to a specific chromosomal location. You can select a whole chromosome or a specific g-band. You can access it by selecting 'Phosphosite' on the select dropdown menu on the top right corner of this page, and then 'Genome Location' on the select dropdown menu that will appear next to the first one.

Phosphoproteomics Data Analysis

Submit a tsv file (other extensions won't work) containing phosphoproteomics data by clicking on the 'Choose File' button of this page.

The file should follow the next format for each inhibitor present on your file, or the analysis won't be correct(the name of the headers does not matter, but the order of the columns does), note that the last two columns are dispensable :

Each substrate in the file is matched to the kinase that phosphorylates it by searching JACKY, and the relative kinase activity (RKA) of each kinase is calculated. This information is returned as four tables and two figures, all of which can be downloaded. The first table will appear on the website, and shows the RKA of each kinase, as well as other information on each kinase:

The kinases RKA is also shown on a bar plot.

The second table shows each substrate on the file submitted and the kinase that phosphorylates them, as well as other information on each phosphosite, For some substrates, a kinase match may not be found on JACKY, the third table has the same columns as the second one, but contains only the substrates for which a kinase match was not found, while the fourth table contains only the substrates for which a kinase match was found.

A volcano plot shows the substrates plotted according to their log fold change (x axis) and their negative log10p (y axis).
